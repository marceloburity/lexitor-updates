# lexitor-updates
Canal oficial de distribuição do Lexitor: versões, changelog e arquivos para atualização automática.
Este repositório é o canal oficial de distribuição do Lexitor, contendo:

Instaladores do Windows (Lexitor_Setup_X.Y.exe)

Pacote portátil (lexitor_Portable.zip)

Manifesto de atualização (updates.json) usado pelo Lexitor para detectar e baixar novas versões
